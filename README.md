# 🔑 Password Generator

Live: https://pwrd-generator.netlify.app/

***A password generator, generates two random passwords based on the password size provided by the user and copy them when you click on one of them.***

## ⚒️ Built with:
- HTML 5
- CSS 3
- JavaScript
